# 주간 마케팅 성과 보고서

이번 주 마케팅 성과를 정리하고 다음 주 액션 플랜을 포함한 보고서를 작성한다.

## 입력
`/weekly-report` — 인수 없이 실행하면 데이터 입력 가이드를 먼저 출력한다.
`/weekly-report <데이터 또는 요약>` — 데이터를 함께 제공하면 즉시 분석한다.

## 실행 방법

**데이터가 없는 경우**: 어떤 데이터를 어떤 형식으로 붙여넣으면 되는지 안내한다.

**데이터가 있는 경우**: 아래 형식으로 `WEEKLY-REPORT-[날짜].md` 파일을 생성하고 터미널에 요약을 출력한다.

# 마케팅 콘텐츠 현지화 번역

마케팅 콘텐츠를 단순 번역이 아닌 타깃 언어의 문화와 표현에 맞게 현지화한다.

## 입력
`/translate <언어코드> <번역할 내용>`

언어코드: `en` (영어), `ko` (한국어), `ja` (일본어), `zh` (중국어), `es` (스페인어)

예시:
- `/translate en 지금 바로 무료로 시작하세요. AI가 당신의 마케팅을 혁신합니다.`
- `/translate ko Start your free trial today. No credit card required.`
- `/translate ja 30일 무료체험 신청하기`

## 현지화 원칙

단순 번역과 현지화의 차이:
- **단순 번역**: 단어를 그대로 다른 언어로 옮김
- **현지화**: 문화적 뉘앙스, 관용표현, 현지 마케팅 관습에 맞게 재창작

## 언어별 마케팅 특성

**영어(EN)**
- 직접적이고 행동 중심적 CTA
- 혜택을 수치로 표현
- 긴박감 표현: "Now", "Today", "Don't miss"

**한국어(KO)**
- 존댓말 사용 여부 결정 (브랜드 톤에 따라)
- 감성적 연결 강조
- 신뢰/검증 표현 중요: "검증된", "믿을 수 있는"

**일본어(JA)**
- 정중한 표현 선호
- 과장 표현 자제
- 구체적 수치와 데이터 신뢰도 높음

**중국어(ZH)**
- 번체(대만/홍콩)와 간체(중국 본토) 구분
- 숫자 마케팅 효과적 (8, 88 등 길한 숫자 활용)
- 사회적 증거와 권위 강조

**스페인어(ES)**
- 라틴아메리카 vs 스페인 방언 차이 확인
- 감성적이고 관계 중심적 표현
- 공동체 가치 강조

## 출력 형식

```
## 원본
[원본 텍스트]

## 현지화 번역
[번역 결과]

## 번역 노트
- [원본과 다르게 표현한 부분과 이유]
- [문화적 고려사항]
- [대안 표현 (있는 경우)]
```
